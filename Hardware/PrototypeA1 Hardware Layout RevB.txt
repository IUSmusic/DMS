IUS Official Drum Sequencer
Technical hardware layout revision B

Device overview

The device is a modular hardware sampler and compact DAW workstation with five track lanes, a master section, a touch monitor, stereo track recording, microphone sampling, soundbank based instrument playback, external monitor support, and removable storage expansion.

Revision B replaces the earlier single DSP focused layout with a system that separates application UI work from deterministic real time audio work.

Real time rule
Playback and record buffers used in the hardest audio path must remain memory backed during active operation
Direct removable media reads must not sit in the tightest deterministic playback path

Master bus and track routing

Per track path
Source select to track engine to optional track processing to track record tap to stereo track out

Master path
All track stereo outs to stereo master bus to master effects to master recorder tap to DAC and outputs

Routing rules
Track Rec prints the selected track behavior in stereo
Track Mic Rec samples from microphone while held
Master Mix Rec captures the final stereo mix
Uploaded full recordings can be assigned as waveform tracks
