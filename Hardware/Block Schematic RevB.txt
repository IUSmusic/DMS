System intent

Open source hardware workstation focused on high quality sampling, soundbank playback, stereo track printing, microphone sampling, touch based DAW editing, and reliable studio grade operation at 24 bit 48 kHz minimum. The architecture must remain responsive under heavy use and must support future timeline editing, waveform navigation, theme changes, external storage, and external display output.

                       AUDIO I O AND ANALOG BOARD

   Input side
   Mic input path or paths to low noise preamp to protection to filter to ADC input routing
   Stereo line input pair 1 left and right to protection to buffer to anti alias filter
   Stereo line input pair 2 left and right to protection to buffer to anti alias filter
   Stereo line input pair 3 left and right to protection to buffer to anti alias filter

   Converter and clock section
   High quality stereo or multichannel codec section
   Clean low jitter master audio clock
   Separate analog and digital rails

   Output side
   DAC output to output filter to line driver to stereo master out left and right
   DAC output to headphone amplifier to headphone out
   Optional balanced output stage in later revisions

   Routing notes
   Track sources are digitized or generated in the digital audio engine
   Track outputs sum to the stereo master bus in the digital domain
   Master recorder taps the stereo master bus
   Optional monitor taps may be exposed pre or post master effects

Storage and connectivity domain

   Internal storage
   Boot storage for compute module
   Project cache and system files

   Removable storage
   More than one micro SD card slot
   One slot can be dedicated to project and recording media
   One slot can be dedicated to sample and soundbank library media

   External storage support
   USB mass storage support including hard drive or SSD by folder structure
   App level storage indexer must understand approved project and library layout

   External ports
   Two USB C ports minimum
   USB A host ports in addition to current ports
   HDMI output
   Existing audio and power ports retained unless revised elsewhere

Audio flow

   Source types
   Soundbank instrument engine
   Sample trigger engine
   Uploaded backing track audio
   Mic input capture
   External audio input

   Per track path
   source select to track engine to track effects or inserts to track recorder tap to track stereo output

   Master path
   all track stereo outputs to stereo master bus to master effects to master recorder tap to DAC and monitor outputs

   Recorder definitions
   Track REC records track behavior and output in stereo without mic unless explicitly sourced that way
   MIC REC on a track records microphone sampling while the user holds the button
   MIX REC on master records the full stereo master mix including all active tracks and instrument playback

Monitor behavior rules

   The monitor is a DAW waveform display not an oscilloscope and not an EQ meter
   The monitor frame remains fixed in size
   Track lanes compress vertically when more tracks are active or engaged
   Track colors are fixed by track number
   Track 1 red
   Track 2 blue
   Track 3 white
   Track 4 red
   Track 5 blue
   Engaged but silent tracks show a flat line
   Recording writes waveform from left to right
   Completed waveforms remain visible for later editing and punch in workflows

Analog versus digital split

   Analog domain
   Mic preamps
   Input protection and buffering
   Anti alias and reconstruction filtering
   Output line driving
   Headphone amplification
   Clean analog power rails and grounding

   Digital domain
   Sequencing and scheduling
   Sample and soundbank playback
   Touch UI and themes
   Waveform rendering and editing
   Track and master routing
   Storage indexing and project management
   Grid and snap editing logic
   HDMI and external display handling

Open source structure recommendation

   Publish by module
   Main compute board
   Real time audio engine board
   Audio I O and analog board
   Control surface board
   Display and touch board

   Publish with
   inter board connector definitions
   power tree
   clock tree
   bus map
   firmware event map
   storage folder specification

Performance target

   Minimum studio target
   24 bit 48 kHz

   Future stretch target
   24 bit 96 kHz if validated without breaking responsiveness

   Core rule
   Active audio must be RAM buffered in the real time path
   Direct removable media access must never sit in the hardest real time playback path
